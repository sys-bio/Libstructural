\subsection{Detailed Description}
\{core\} A token from \hyperlink{_formula_tokenizer_8h_a85f75f0f003ba89ae22163cef7dacb1d}{Formula\+Tokenizer\+\_\+next\+Token()}.

A \hyperlink{struct_token__t}{Token\+\_\+t} token has a {\ttfamily type} and a {\ttfamily value}. The {\ttfamily value} field is a union of different possible members; the member that holds the value for a given token (and thus the name of the member that is to be accessed programmatically) depends on the value of the {\ttfamily type} field. The following table lists the possible scenarios\+:

\begin{center} \tabulinesep=1mm
\begin{longtabu} spread 0pt [c]{*{2}{|X[-1]}|}
\hline
\rowcolor{\tableheadbgcolor}\textbf{ Value of the field {\ttfamily type}}&\textbf{ Member within {\ttfamily value} to use }\\\cline{1-2}
\endfirsthead
\hline
\endfoot
\hline
\rowcolor{\tableheadbgcolor}\textbf{ Value of the field {\ttfamily type}}&\textbf{ Member within {\ttfamily value} to use }\\\cline{1-2}
\endhead
{\ttfamily T\+T\+\_\+\+N\+A\+ME}&{\ttfamily name} \\\cline{1-2}
{\ttfamily T\+T\+\_\+\+I\+N\+T\+E\+G\+ER}&{\ttfamily integer} \\\cline{1-2}
{\ttfamily T\+T\+\_\+\+R\+E\+AL}&{\ttfamily real} \\\cline{1-2}
{\ttfamily T\+T\+\_\+\+R\+E\+AL}\+\_\+E&{\ttfamily real} and {\ttfamily exponent} \\\cline{1-2}
{\itshape Anything else}&{\ttfamily ch} \\\cline{1-2}
\end{longtabu}
\end{center} 

If this token encodes a real number in e-\/notation, {\ttfamily type} will be {\ttfamily T\+T\+\_\+\+R\+E\+A\+L\+\_\+E} instead of {\ttfamily T\+T\+\_\+\+R\+E\+AL}. The field {\ttfamily value.\+real} will then contain the mantissa, and the separate field named {\ttfamily exponent} will contain (can you guess?) the exponent. For example, if we have a pointer to a structure named {\ttfamily t}, then the representation for "{\ttfamily 1.\+2e3}" will be\+: \begin{DoxyVerb}t->type       = TT_REAL_E;
t->value.real = 1.2;
t->exponent   = 3;
\end{DoxyVerb}


When the {\ttfamily type} has a value of {\ttfamily T\+T\+\_\+\+U\+N\+K\+N\+O\+WN}, the field {\ttfamily ch} will contain the unrecognized character. When the type is {\ttfamily T\+T\+\_\+\+E\+ND}, the field {\ttfamily ch} will contain {\ttfamily \textquotesingle{}\textbackslash{}0\textquotesingle{}}. For all others, the {\ttfamily value.\+ch} will contain the corresponding character.
