\subsection{Detailed Description}
basic functions for steady state flux balance analysis of S\+B\+ML files 

\begin{DoxyParagraph}{}
This file states flux balance analysis of S\+B\+ML models (or Stoichiometry matrices) along with flux constraints as linear programming question, which is then solved using lpsolve (\href{http://sf.net/projects/lpsolve}{\tt http\+://sf.\+net/projects/lpsolve}).
\end{DoxyParagraph}

Frank T. Bergmann (\href{mailto:fbergman@u.washington.edu}{\tt fbergman@u.\+washington.\+edu}) 
\end{DoxyAuthor}
