\subsection{Detailed Description}
C++ I/O streams interface to the zip/unzip functions in Minizip. 

\begin{DoxyAuthor}{Author}
Akiya Jouraku
\end{DoxyAuthor}
Most of the code (except for zip/unzip specific code) is based on \hyperlink{zfstream_8h}{zfstream.\+h} implemented by Ludwig Schwardt \href{mailto:schwardt@sun.ac.za}{\tt schwardt@sun.\+ac.\+za} original version by Kevin Ruland \href{mailto:kevin@rodin.wustl.edu}{\tt kevin@rodin.\+wustl.\+edu} \hyperlink{zfstream_8h}{zfstream.\+h} is contained in the contributed samples in zlib version 1.\+2.\+3 (\href{http://www.zlib.net}{\tt http\+://www.\+zlib.\+net}).

The following zip/unzip specific functions are implemented based on the code in minizip.\+c and miniunz.\+c contained in Minizip version 1.\+01e (\href{http://www.winimage.com/zLibDll/minizip.html}{\tt http\+://www.\+winimage.\+com/z\+Lib\+Dll/minizip.\+html}) implemented by Gilles Vollant.
