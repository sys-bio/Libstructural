Detailed Description

bzip2 file stream buffer class.

This class implements basic_filebuf for bzip2 files. It doesn't yet support seeking (allowed by zlib but slow/limited), putback and read/write access (tricky). Otherwise, it attempts to be a drop-in replacement for the standard file streambuf.
