Detailed Description

C++ I/O streams interface to the zip/unzip functions in Minizip.

Author
Akiya Jouraku

Most of the code (except for zip/unzip specific code) is based on zfstream.h implemented by Ludwig Schwardt schwa.nosp@m.rdt@.nosp@m.sun.a.nosp@m.c.za original version by Kevin Ruland kevin.nosp@m.@rod.nosp@m.in.wu.nosp@m.stl..nosp@m.edu zfstream.h is contained in the contributed samples in zlib version 1.2.3 (http://www.zlib.net).

The following zip/unzip specific functions are implemented based on the code in minizip.c and miniunz.c contained in Minizip version 1.01e (http://www.winimage.com/zLibDll/minizip.html) implemented by Gilles Vollant.

zipFile zipopen (const char* path, const char* filenameinzip, int append); int zipclose(zipFile file); int zipwrite(zipFile file, voidp buf, unsigned len); unzFile unzipopen (const char* path); int unzipclose(unzFile file); int unzipread (unzFile file, voidp buf, unsigned len);
