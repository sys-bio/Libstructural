doc_eventassignment_units Class Reference

#include <EventAssignment.h>

Detailed Description

The units are calculated based on the mathematical expression in the EventAssignment and the model quantities referenced by <ci> elements used within that expression. The method getDerivedUnitDefinition() returns the calculated units, to the extent that libSBML can compute them.
